# \# 📊 App Insights Unlocked: Google Play Store Data Analysis

# 

### \## 📌 Project Overview



This project focuses on analyzing Google Play Store app data to uncover insights that can help improve app development strategies, enhance user experience, and drive business growth.



The dataset includes information about app ratings, reviews, installs, size, price, and categories.



\---



\## 🎯 **Problem Statement**



The objective of this project is to:



\* Identify key factors contributing to app success

\* Analyze user behavior and preferences

\* Explore relationships between installs, ratings, and pricing

\* Provide actionable insights for business decision-making



\---



\## 📂 **Dataset**



\* Source: Google Play Store dataset (Kaggle)

\* Key features:



&#x20; \* App Name

&#x20; \* Category

&#x20; \* Rating

&#x20; \* Reviews

&#x20; \* Installs

&#x20; \* Price

&#x20; \* Size

&#x20; \* Content Rating



\---



\## 🧹 **Data Cleaning \& Preprocessing**



The dataset was cleaned and prepared using the following steps:



\* Removed duplicate records

\* Converted data types (Rating, Reviews, Installs, Price)

\* Cleaned "Installs" column (removed + and commas)

\* Cleaned "Price" column (removed $ symbol)

\* Converted "Size" into MB format

\* Converted "Last Updated" to datetime format

\* Handled missing values (filled or removed)

\* Standardized text fields (Category, Genres)



\---



\## 🛠 **Tools \& Technologies Used**



\* Microsoft Excel (Data Cleaning \& Analysis)

\* Power BI (Dashboard \& Visualization)

\* Python (Optional data preprocessing)

\## 📊 **Exploratory Data Analysis (EDA)**



EDA was performed to understand:



\* Distribution of app ratings

\* Popular app categories

\* Relationship between installs and ratings

\* Free vs Paid app trends

\* User engagement through reviews



\---



\## 📈 **Dashboard Highlights**



The Power BI dashboard includes:



\* KPI Cards (Average Rating, Total Apps, Installs, Reviews)

\* Category-wise Install Analysis

\* Free vs Paid App Distribution

\* Content Rating Analysis

\* Rating vs Installs Scatter Plot

\* Top Apps by Installs and Reviews



\---



\## 🔍 **Key Insights**



\* Free apps dominate the market (\~90%)

\* Games and Communication categories have the highest installs

\* Ratings are not strongly correlated with installs

\* High number of reviews indicates strong user engagement

\* Most apps target the "Everyone" content rating



\---



\## 💡 **Business Recommendations**



\* Adopt a freemium pricing model

\* Focus on high-demand categories like Games and Tools

\* Improve user experience to increase ratings

\* Regularly update apps to maintain engagement

\* Optimize app size for better performance

\* Ensuring data consistency across columns
